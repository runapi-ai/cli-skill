# Changelog

## [v0.2.10](https://github.com/runapi-ai/cli-skill/releases/tag/v0.2.10) - 2026-07-17

### Changed
- Document JSON API key discovery, explicit stable-ID selection, strict project configuration, and per-key listener secret retrieval.
- Guide agents through login-required, missing-selection, cross-member configuration, and unusable-key recovery without silently selecting another key.
- Require listener users upgrading from the previous behavior to update the CLI, log in again, replace the webhook secret, and restart listeners.
